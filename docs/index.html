locksmithx

The professional, cross-platform file locking utility for Node.js

GitHub npm

Features

  • Cross-platform: Windows, Linux, macOS, and network filesystems
  • Atomic, reliable file locking using mkdir strategy
  • Staleness detection & automatic lock refreshing
  • Async & sync APIs for maximum flexibility
  • Customizable retry, staleness, and update intervals
  • Handles process crashes and cleans up locks on exit
  • Lightweight, zero dependencies for core functionality

Installation

npm install locksmithx

Quick Start

const locksmith = require('locksmithx');

(async () => {
  const release = await locksmith.lock('some/file');
  try {
    // Do something while the file is locked
  } finally {
    await release();
  }
})();

Why locksmithx?

locksmithx is designed for modern Node.js applications that need robust, reliable, and cross-platform file locking. It is ideal for distributed systems, build tools, and any scenario where safe file access is critical.
